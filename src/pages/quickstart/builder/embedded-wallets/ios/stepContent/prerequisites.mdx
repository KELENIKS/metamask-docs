### Prerequisites

- iOS 14+
- Xcode 12+
- Swift 5.x

:::tip build locally

To build the project locally:

- Clone the MetaMask Embedded Wallets Swift/iOS quickstart application:

  ```shell npm2yarn
  npx degit Web3Auth/web3auth-ios-examples/ios-quick-start w3a-ios-quick-start
  ```

- Open the project in Xcode from the `w3a-ios-quick-start` directory and
    [build the app](https://developer.apple.com/documentation/xcode/building-and-running-an-app).

:::
